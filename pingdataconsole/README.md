# PingDataConsole

Remember to add the repo before installing the chart
```shell
helm repo add pingidentity-pc https://patrickcping.github.io/ping-identity-helm-charts-repo/
```

## Running

```shell
helm install \
pingdataconsole --debug \
pingidentity-pc/pingdataconsole
```
